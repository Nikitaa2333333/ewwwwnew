# /push — Push to GitHub

Пушит текущие изменения на GitHub.

## Поведение

1. Показать `git status` — что изменилось
2. Спросить у пользователя сообщение коммита (или предложить своё на русском)
3. `git add -A`
4. `git commit -m "<сообщение>"`
5. `git push`
6. Сообщить результат

## Правила

- Никогда не пушить без явного запроса пользователя
- Не использовать `--force`
- Сообщение коммита — на русском, лаконичное, описывает суть изменений
