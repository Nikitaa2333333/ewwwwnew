# Ривер Лофт — Landing

Одностраничный лендинг премиум-площадки для мероприятий. Акцент — на визуал, выразительную типографику и плавные анимации. Никаких лишних абстракций.

## Стек

- **React 19** + **Vite 6** + **TypeScript**
- **Tailwind CSS v4** (конфигурация через `@theme` в `index.css`)
- **motion/react** (Framer Motion) — единственный инструмент для анимаций

## Команды

```bash
npm run dev      # dev-сервер на :3000
npm run build    # production-сборка в /dist
npm run preview  # превью сборки
npm run lint     # tsc --noEmit
```

## Структура

- `src/App.tsx` — все компоненты лендинга в одном файле
- `src/index.css` — дизайн-токены через `@theme`, базовые стили
- `public/` — изображения (WebP)
